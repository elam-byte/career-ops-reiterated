# Mode: training — Training & Certification Evaluation

For each course/cert the candidate asks about, evaluate 6 dimensions:

| Dimension | What it measures |
|-----------|-----------------|
| North Star alignment | Does it move closer to or further from the target? |
| Recruiter signal | What do hiring managers think when they see this on a CV? |
| Time and effort | Weeks × hours/week |
| Opportunity cost | What can't be done during that time? |
| Risks | Outdated content? Weak brand? Too basic? |
| Portfolio deliverable | Does it produce a demonstrable artefact? |

## Verdicts

- **DO IT** → 4–12 week plan with weekly deliverables and a scoreboard
- **DON'T DO IT** → better alternative with justification
- **DO IT WITH TIMEBOX** (max X weeks) → condensed plan, essentials only

## Priority by archetype

Detect the user's current target archetype (from `_shared.md` + `_profile.md`) and apply the relevant priority stack.

**AI Platform / LLMOps / Agentic / AI Solutions Architect:**
1. LLM evals and testing frameworks
2. Observability and monitoring for ML systems
3. Cost / reliability trade-offs in production AI
4. AI governance and responsible AI
5. Enterprise AI architecture patterns

**Program Manager / Technical Program Manager:**
1. SAFe / Agile at scale certifications (SAFe6, PMP)
2. Financial governance and portfolio stage-gates
3. Risk management and RAID frameworks
4. Stakeholder communication and executive reporting
5. Tools: Jira Advanced Roadmaps, MS Project, Smartsheet

**Product Manager / Technical Product Manager:**
1. Product discovery and user research methods
2. Metrics and experimentation (A/B testing, HEART framework)
3. Roadmap prioritisation frameworks (RICE, OKRs, Kano)
4. Stakeholder alignment and go-to-market
5. Domain depth in the target product area

**Solution Architect / Systems Architect:**
1. Cloud architecture certifications (Azure Solutions Architect Expert, AWS SAA)
2. MBSE and ArchiMate / TOGAF
3. Safety-critical systems (ASPICE, ISO 26262) if not already certified
4. Security architecture (CISSP, SABSA) if relevant to target roles
5. Domain depth: automotive, embedded, enterprise, or AI platform
